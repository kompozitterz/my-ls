# My-ls
Recréation de la commande Unix `ls` en Go.

## Fonctionnalités
- Liste les fichiers et répertoires.
- Supporte les flags : 
    * `-l`:  Format détaillé (long format), 
    * `-R`:  Récursivité, 
    * `-a`:  Afficher tous les fichiers, 
    * `-r`:  Inverser l'ordre, 
    * `-t`:  Trier par date de modification.

## Usage 

> Compilation 

```bash
go build -o my-ls
```

> Script

```bash
./script.sh
```
